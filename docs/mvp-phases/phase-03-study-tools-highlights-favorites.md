# Phase 3: 공부 기능, 강조, 인용 구절 보관함

## 목표

사용자가 중요한 구절을 형광펜처럼 표시하고, 자주 인용하는 구절을 개인 보관함에 저장하며, 태그와 복사 기능으로 성경 공부 자료를 관리할 수 있게 한다. v0에서는 mock 사용자와 local/mock storage 기준으로 구현한다.

## 구현 체크리스트

### 1. 데이터 구조

- [ ] `highlights` 테이블 또는 동등한 저장소를 만든다.
- [ ] `favorite_verses` 테이블 또는 동등한 저장소를 만든다.
- [ ] `tags` 테이블 또는 동등한 저장소를 만든다.
- [ ] `favorite_verse_tags` 연결 테이블 또는 동등한 구조를 만든다.
- [ ] 강조 색상 enum 또는 검증 규칙을 정의한다.
- [ ] 한 사용자가 같은 구절을 중복 저장할 때의 정책을 정한다.
- [ ] 태그 이름은 사용자별로 중복 정책을 정한다.

### 2. 강조 구절 API

- [ ] `GET /api/me/highlights`를 구현한다.
- [ ] `POST /api/me/highlights`를 구현한다.
- [ ] `PATCH /api/me/highlights/:id`를 구현한다.
- [ ] `DELETE /api/me/highlights/:id`를 구현한다.
- [ ] 강조 색상과 메모 입력을 검증한다.
- [ ] 사용자가 소유한 강조만 수정/삭제할 수 있게 한다.
- [ ] 강조 목록 조회 시 성경 권/장/절과 본문을 함께 반환한다.

### 3. 인용 구절 API

- [ ] `GET /api/me/favorite-verses`를 구현한다.
- [ ] `POST /api/me/favorite-verses`를 구현한다.
- [ ] `PATCH /api/me/favorite-verses/:id`를 구현한다.
- [ ] `DELETE /api/me/favorite-verses/:id`를 구현한다.
- [ ] 구절 제목, 메모, 사용 횟수 필드를 저장한다.
- [ ] 저장 구절 조회 시 성경 위치와 본문을 함께 반환한다.
- [ ] 복사 또는 사용 시 `usage_count` 증가 정책을 정한다.

### 4. 태그 API

- [ ] `GET /api/me/tags`를 구현한다.
- [ ] `POST /api/me/tags`를 구현한다.
- [ ] `PATCH /api/me/tags/:id`를 구현한다.
- [ ] `DELETE /api/me/tags/:id`를 구현한다.
- [ ] 인용 구절에 태그를 연결하고 해제하는 로직을 구현한다.
- [ ] 태그 삭제 시 연결 데이터를 어떻게 처리할지 정한다.

### 5. 성경 리더 연동

- [ ] 절 클릭 또는 선택 상태를 구현한다.
- [ ] 절 선택 메뉴에 강조, 인용 저장, 복사 버튼을 배치한다.
- [ ] 강조 색상 선택 UI를 만든다.
- [ ] 이미 강조된 구절은 리더에서 색상으로 표시한다.
- [ ] 강조 색상 변경과 해제를 리더에서 처리한다.
- [ ] 인용 구절 저장 폼을 리더에서 열 수 있게 한다.
- [ ] 저장 완료 후 사용자에게 명확한 상태 변화를 보여준다.

### 6. 강조 구절 화면

- [ ] 강조 구절 목록 페이지를 만든다.
- [ ] 색상 필터를 제공한다.
- [ ] 성경 권 필터를 제공한다.
- [ ] 메모를 표시하고 수정할 수 있게 한다.
- [ ] 강조 해제를 목록에서 처리할 수 있게 한다.
- [ ] 목록 항목 클릭 시 해당 성경 장과 절로 이동한다.

### 7. 인용 구절 보관함

- [ ] 저장한 구절 목록 페이지를 만든다.
- [ ] 제목 또는 별칭을 표시한다.
- [ ] 태그 목록을 표시한다.
- [ ] 태그 추가/삭제 UI를 제공한다.
- [ ] 최근 저장순 정렬을 제공한다.
- [ ] 성경 순서 정렬을 제공한다.
- [ ] 자주 사용순 정렬을 제공한다.
- [ ] 삭제 기능을 제공한다.

### 8. 구절 복사

- [ ] 구절 본문과 위치를 함께 복사하는 기능을 만든다.
- [ ] 기본 복사 형식은 PRD 예시를 따른다.
- [ ] 복사 성공/실패 상태를 표시한다.
- [ ] 리더, 강조 목록, 인용 구절 보관함에서 복사할 수 있게 한다.
- [ ] 모바일 브라우저에서 Clipboard API fallback을 검토한다.

### 9. 검증

- [ ] 요한복음 3장 16절을 노란색으로 강조할 수 있다.
- [ ] 강조한 구절이 새로고침 후에도 유지된다.
- [ ] 강조 색상을 변경할 수 있다.
- [ ] 강조 구절 목록에서 해당 구절을 다시 찾을 수 있다.
- [ ] 로마서 10장 9절을 인용 구절로 저장할 수 있다.
- [ ] 태그 "구원", "복음"을 연결할 수 있다.
- [ ] 저장한 구절을 PRD 복사 형식으로 복사할 수 있다.
- [ ] v0에서는 mock 사용자 기준으로 강조와 인용 구절이 유지된다.

## 수용 기준

- 사용자는 절 단위로 구절을 강조하고 색상을 바꿀 수 있다.
- 강조한 구절은 다시 접속해도 유지된다.
- 사용자는 원하는 구절을 인용 구절 보관함에 저장할 수 있다.
- 저장된 구절은 태그와 함께 관리할 수 있다.
- 구절 복사 시 본문과 성경 위치가 함께 복사된다.
- v0에서는 강조와 인용 데이터가 mock 사용자 기준으로 유지되고, 실제 출시 전에는 계정별 분리를 적용한다.
